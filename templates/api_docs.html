GET /api/events/{event_id}

Retrieve details of a specific event by ID.

Parameters:

Parameter Type Description
event_id String Unique identifier of the event

Response Example:

{
  "success": true,
  "event": {
    "id": "abc123",
    "name": "Tech Summit 2025",
    "description": "Annual technology conference featuring industry leaders",
    "date": "2025-03-15",
    "time": "10:00",
    "venue": "Qatar Convention Centre",
    "price": 75.00,
    "category": "Conference",
    "total_seats": 200,
    "available_seats": 150
  }
}

Error Response (404):

{
  "success": false,
  "error": "Event not found"
}

POST /api/chat

Send a message to the AI customer support chatbot.

Request Body:

{
  "message": "How do I book an event?"
}

Response Example:

{
  "success": true,
  "message": "To book an event: 1. Browse events on our Events page..."
}

Error Response (400):

{
  "success": false,
  "error": "No message provided"
}

HTTP Status Codes

Code Meaning Description
200 OK Request successful
400 Bad Request Invalid request parameters
404 Not Found Resource not found
500 Server Error Internal server error
